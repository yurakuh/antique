# Atelier Nordique — GitHub Pages

Если вы уже включили **GitHub Pages**, сайт открывается по адресу:

- `https://<ваш-логин>.github.io/<имя-репозитория>/`

Для этого репозитория обычно это будет:

- `https://<ваш-логин>.github.io/antique/`

## Что нужно сделать один раз

1. В GitHub: **Settings → Pages**.
2. В разделе **Source** выберите: **GitHub Actions**.
3. Убедитесь, что основная ветка — `main`.
4. Запушьте изменения в `main`.

После пуша GitHub запустит workflow `Deploy Next.js static site to GitHub Pages`.

## Как проверить, что деплой прошёл

1. Откройте вкладку **Actions** в репозитории.
2. Дождитесь, пока workflow завершится со статусом ✅.
3. Перейдите в **Settings → Pages** — там будет точная ссылка на опубликованный сайт.

## Если открывается 404

- Проверьте, что workflow завершился успешно.
- Проверьте, что в Pages выбран именно **GitHub Actions**, а не branch deploy.
- Подождите 1–3 минуты после успешного деплоя: иногда CDN обновляется не мгновенно.
